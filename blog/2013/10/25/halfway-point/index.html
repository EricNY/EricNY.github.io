Halfway point is here, this is the end of week 6. We started with frontend development this week which meant brushing up on our HTML and CSS and getting into Javascript.

I wrote a lot of code using JS in preparing for this course but it certainly took some time to get used to it again since we’ve been focused on using Ruby so much these past few weeks. The principles are still the same no matter what language you use like recursion, iteration, various data structures, etc… they just each have their own way of implementing them.

I am getting more comfortable using various tools in my “box”. Apparently that is the cool programmer way to refer to my computer! Lol. I have a lot more to learn though regarding my development environment, even my text editor has more tricks up its sleeve that I still need to get familiar with.

This week we started Fullstack’s “CTO Program” talks. We had a group discussion with Dave where we talked about concepts from Eric Reis’ book The Lean Startup. It is a very exciting time to be a web developer since we can start a business with basically nothing. Of course scaling is an whole other matter but it has been hammered home to us that we should not focus on scaling issues until the time comes where it is necessary. Until then dont let it get in the way of trying out new ideas.
